# HopeBridge Foundation

Client-ready NGO donation website demo built with Next.js App Router, TypeScript, Tailwind CSS, and reusable components.

## Setup

```bash
npx create-next-app@latest hopebridge-foundation
cd hopebridge-foundation
npm.cmd install lucide-react
npm.cmd run dev
```

## Payment integration

The donation form is demo-only. A live version should connect to a secure API route and payment provider such as Stripe, PayPal, Flutterwave, Pesapal, DPO, Selcom, or mobile money gateway.
